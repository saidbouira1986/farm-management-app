# تطبيق إدارة المزرعة - Farm Management App

## نظرة عامة
تطبيق Android شامل لإدارة المزارع المتخصصة في تربية الأغنام والخضروات، مع تتبع كامل للمصاريف والأرباح.

## المميزات الرئيسية

### 1. نظام المستخدمين
- تسجيل مستخدم جديد
- تسجيل دخول آمن
- ملف شخصي للمزرعة

### 2. إدارة الأغنام
- إضافة وتحديث بيانات الأغنام
- تتبع: الجنس، العمر، الحالة الصحية، الوزن، السلالة
- تتبع سعر الشراء والقيمة السوقية
- قائمة الأغنام النشطة والمباعة

### 3. إدارة المصاريف
- تسجيل مصاريف العلف
- تكاليف الأدوية والعلاجات
- رواتب العمال
- مصاريف أخرى متنوعة
- تصنيف وتحليل المصاريف

### 4. إدارة الدخل
- تسجيل بيع الأغنام
- حساب الربح التلقائي
- تتبع سجل المبيعات
- إحصائيات الدخل

### 5. إدارة الصحة
- تسجيل الأمراض والمشاكل الصحية
- تتبع العلاجات والتكاليف
- حالة التعافي
- سجل الأمراض السابقة

### 6. لوحة المعلومات
- عرض الإحصائيات العامة
- الأرباح والخسائر
- عدد الأغنام
- أداء المزرعة
- رسوم بيانية وتقارير

## المتطلبات التقنية
- Android 5.0 (API 21) فما فوق
- Java 11+
- Gradle 7.4.2+

## التكنولوجيا المستخدمة
- **قاعدة البيانات**: Room Database
- **التشفير**: SHA-256 للكلمات المرورية
- **الواجهات**: XML Layout مع MaterialDesign
- **البناء**: Gradle

## البنية المعمارية
```
app/src/main/java/com/farm/management/
├── database/          # قاعدة البيانات
├── models/            # نماذج البيانات
├── dao/               # Data Access Objects
├── ui/                # الأنشطة والواجهات
├── repository/        # طبقة المستودع
└── util/              # الأدوات المساعدة
```

## اللغات المدعومة
- العربية (الافتراضية)
- الإنجليزية

## كيفية البناء والتثبيت

### البناء
1. افتح المشروع في Android Studio
2. انتظر تحميل جميع المكتبات
3. اختر Build > Build Bundle(s) / APK(s) > Build APK(s)
4. سيظهر ملف APK في `app/build/outputs/apk/debug/`

### التثبيت على الجهاز
1. انقل ملف APK إلى جهازك الذكي
2. اضغط عليه لتثبيت التطبيق
3. افتح التطبيق وسجل حساباً جديداً
